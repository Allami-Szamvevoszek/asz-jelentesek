# Az Állami Számvevőszék (ÁSZ) jelentései (markdown+jpeg)

Ez a repository az [Állami Számvevőszék](https://www.asz.hu/) jelentéseit tartalmazza 1989-től (a második világháború utáni újáalakulása óta), feldolgozható formában.

Az eredeti, PDF-be-scan-elt dokumentumokat *Mistral OCR*-el alakítottuk szöveggé, majd egy hibajavító fázison estek át Google *gemini-1.5-flash* modell segítségével.

FIGYELEM! Az anyag **hibákat tartalmaz**, elsősorban a mellékletekben és alátámasztó anyagokban; de bárhol a szövegben előfordulhatnak (és elő is fordulnak!) scannelési, OCR-ezési, AI "képzelgési", vagy bármilyen hibák.

Az eredeti scannelt PDF-ek megtalálhatóak [ebben a másik repository-ban](https://github.com/Allami-Szamvevoszek/asz-jelentesek-pdf).

# Reports of the Hungarian State Audit Office (ÁSZ)

This repository contains the reports of the [State Audit Office](https://www.asz.hu/en/main) of Hungary, in Hungarian, since 1989 (its re-establishment after World War II), in a machine-readable format.

The original, scanned-to-PDF documents were processed with *Mistral OCR*, then a fix-up phase was done with Google *gemini-1.5-flash*.

WARNING! The material **contains errors**, primarily in the appendices and attached support materials; but scanning, OCR, AI imagination and other errors can happen anywhere (and they do).

The original scanned PDFs can be found [in this other repository](https://github.com/Allami-Szamvevoszek/asz-jelentesek-pdf).
